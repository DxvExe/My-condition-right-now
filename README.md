🤖 About URA

URA is my biggest project so far.

It's an AI-powered platform built to help students learn more effectively.

Features

- 💬 AI-powered assistant
- 📚 Student-focused learning
- 🔐 Authentication
- ⚡ Fast frontend
- 🌐 Full-stack architecture

One of the proudest moments of my journey was seeing URA reach over 1,000 users.

Watching people actually use something I built was an unforgettable feeling.

But success also taught me one of my biggest lessons.

The backend was hosted on a free Render server. After the traffic spike, it couldn't keep up. Eventually it became unstable and later struggled to handle even four users at the same time.

It was frustrating.

Instead of giving up, I learned about deployment, backend optimization, scalability, caching, and why infrastructure matters just as much as writing code.

Also... many people ask why it's called URA.

The answer is simple.

It's inspired by Uranus, my favorite planet.

Ever since I was little, Uranus has fascinated me, and one of my biggest dreams is to help humanity understand it better. Naming my biggest project after the planet that inspired my curiosity felt right.
